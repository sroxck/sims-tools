### 安装

``` bash
yarn add sims-tools # 或者：npm install sims-tools
```

### 引入全局对象
``` ts
import tools from 'sims-tools'

// 使用
tools.isEmpty(null)
```

### 单独引入函数
``` ts
import { isEmpty } from 'sims-tools'

// 使用
isEmpty(null)
```

### 按需引入
``` ts
import { merge } from 'sims-tools/merge'

// 使用
isEmpty(null)
```
